# Для запуска на Windows:

1. Установить интерпретатор, выбрав опцию "Add Python 3.8 to PATH":

для 64 битной системы: https://www.python.org/ftp/python/3.8.6/python-3.8.6-amd64.exe  
для 32 битной системы: https://www.python.org/ftp/python/3.8.6/python-3.8.6.exe

2. Скачать и распаковать zip архив:  
<p align="center">
  <img src="https://raw.githubusercontent.com/vdevelg/MarkV/devel/man/git_repo.png" alt="git_repo.png">
</p>

3. Выполнить в командной строке:  

> python -m pip install -r requirements.txt

ЛИБО запустить файл:

> install_requirements.cmd

4. Запустить файл:  

> MarkV.pyw

5. Нажать кнопку "Справка"  



# Для обновления:

1. Скачать и распаковать zip архив. (см. пункт 2 установки)  

2. Запустить файл:  

> MarkV.pyw
